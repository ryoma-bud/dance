# DanceConnect（仮）💃🕺

ダンサー向けのSNS兼コミュニティプラットフォーム。
動画の投稿・閲覧、ダンサー同士の交流、レッスン・イベントの募集／応募ができるアプリです。

---

## 📌 概要

ダンサーが自分のパフォーマンス動画を共有し、ファンや仲間とつながれるSNSです。
さらに掲示板機能を通じて、レッスンやイベントの主催・応募が行えます。

---

## 🛠 技術スタック

| 領域       | 使用技術          |
| ---------- | ----------------- |
| フロントエンド | Next.js           |
| バックエンド   | Java（Spring Boot)|
| データベース   | PostgreSQL |
| 認証         | JWT / OAuth |

---

## ✨ 主な機能

### 1. SNS機能

- **メインフィード**
  - Instagramの検索画面のような、動画タイル一覧表示
- **アカウント機能**
  - ユーザー登録 / ログイン
  - プロフィールページ（投稿した動画の一覧表示）
- **インタラクション**
  - いいね
  - フォロー / フォロワー

### 2. 掲示板機能

別タブで提供。以下の2種類の板を用意。

#### 📖 レッスン板
- 開催者がレッスン募集を投稿
- 利用者が参加フォームから応募

#### 🎤 イベント板
タブで分かれた2つの募集形式。

| タブ           | 内容 |
| -------------- | ---- |
| 出演者募集     | 主催がイベントを企画 → 出演者（個人 / チーム）が応募 |
| 観戦者（一般客）募集 | 出演者確定後、一般客が参加応募可能 |

- **チーム応募対応**
  - 出演者がチームの場合、リーダーが応募時にメンバーを記載
- **出演イベント検索**
  - 検索ボックスに出演者名やチームメンバー名を入力すると、出演予定イベントを検索可能

この文書は、ダンサー向けSNS兼コミュニティサービス「DanceConnect（仮）」を、2人で2026-07から2026-12まで開発するための設計書です。

#### 2. サービス概要

#### 2-1. サービス名
- **仮名称**：DanceConnect

#### 2-2. サービス内容
DanceConnectは、ダンサー向けのSNS兼コミュニティプラットフォームです。

ユーザーは動画を投稿して他のダンサーと交流できるほか、掲示板を使って以下のような募集や応募ができます。

- イベント募集
- レッスン募集
- チームメンバー募集

また、投稿した動画や掲示板の投稿は、全体一覧だけでなく、そのユーザーのプロフィール画面からも見られるようにします。

#### 2-3. 想定ユーザー
- ダンス動画を投稿したい人
- 他のダンサーとつながりたい人
- レッスン参加者を募集したい人
- イベント出演者や観客を募集したい人
- チームメンバーを募集したい人

#### 3. 開発体制

#### 3-1. 体制
- 開発人数は2人
- フロントエンド担当、バックエンド担当で完全に分けない
- 機能ごとに担当を持ち、その担当者がフロントエンドとバックエンドの両方を実装する

#### 3-2. 進め方
例えば、以下のように進めます。

- 担当A：会員登録機能を担当
    - 画面作成
    - API接続
    - バックエンド実装
    - DB保存まで担当
- 担当B：動画投稿機能を担当
    - 投稿画面
    - API設計
    - バックエンド処理
    - 保存処理まで担当

この形にすると、各機能の流れを理解しやすく、2人ともサービス全体への理解が深まります。
