# DanceConnect（仮）💃🕺

ダンサー向けのSNS兼コミュニティプラットフォーム。
動画の投稿・閲覧、ダンサー同士の交流、レッスン・イベントの募集／応募ができるアプリです。

---

## 📌 概要

ダンサーが自分のパフォーマンス動画を共有し、ファンや仲間とつながれるSNSです。
さらに掲示板機能を通じて、レッスンやイベントの主催・応募が行えます。

---

## 🛠 技術スタック

| 領域       | 使用技術          |
| ---------- | ----------------- |
| フロントエンド | Next.js           |
| バックエンド   | Java（Spring Boot)|
| データベース   | PostgreSQL |
| 認証         | JWT / OAuth |

---

## ✨ 主な機能

### 1. SNS機能

- **メインフィード**
  - Instagramの検索画面のような、動画タイル一覧表示
- **アカウント機能**
  - ユーザー登録 / ログイン
  - プロフィールページ（投稿した動画の一覧表示）
- **インタラクション**
  - いいね
  - フォロー / フォロワー

### 2. 掲示板機能

別タブで提供。以下の2種類の板を用意。

#### 📖 レッスン板
- 開催者がレッスン募集を投稿
- 利用者が参加フォームから応募

#### 🎤 イベント板
タブで分かれた2つの募集形式。

| タブ           | 内容 |
| -------------- | ---- |
| 出演者募集     | 主催がイベントを企画 → 出演者（個人 / チーム）が応募 |
| 観戦者（一般客）募集 | 出演者確定後、一般客が参加応募可能 |

- **チーム応募対応**
  - 出演者がチームの場合、リーダーが応募時にメンバーを記載
- **出演イベント検索**
  - 検索ボックスに出演者名やチームメンバー名を入力すると、出演予定イベントを検索可能

## 🧩 データモデル（簡易案）

### User（ユーザー）
| カラム      | 型      | 説明           |
| ----------- | ------- | -------------- |
| id          | bigint  | 主キー         |
| name        | string  | 表示名         |
| email       | string  | ログイン用     |
| profile     | text    | 自己紹介       |

### Video（動画）
| カラム     | 型      | 説明             |
| ---------- | ------- | ---------------- |
| id         | bigint  | 主キー           |
| user_id    | bigint  | 投稿者           |
| url        | string  | 動画URL          |
| likes      | int     | いいね数         |

### Follow（フォロー関係）
| カラム        | 型     | 説明           |
| ------------- | ------ | -------------- |
| follower_id   | bigint | フォローする人 |
| following_id  | bigint | される人       |

### LessonPost（レッスン募集）
| カラム      | 型     | 説明       |
| ----------- | ------ | ---------- |
| id          | bigint | 主キー     |
| host_id     | bigint | 開催者     |
| title       | string | タイトル   |
| detail      | text   | 詳細       |

### LessonApply（レッスン応募）
| カラム      | 型     | 説明       |
| ----------- | ------ | ---------- |
| id          | bigint | 主キー     |
| lesson_id   | bigint | 対象レッスン |
| user_id     | bigint | 応募者     |

### Event（イベント）
| カラム      | 型     | 説明       |
| ----------- | ------ | ---------- |
| id          | bigint | 主キー     |
| host_id     | bigint | 主催者     |
| title       | string | タイトル   |
| detail      | text   | 詳細       |
| status      | enum   | 出演者募集中 / 一般客募集中 / 終了 |

### EventPerformer（出演者応募）
| カラム       | 型     | 説明                   |
| ------------ | ------ | ---------------------- |
| id           | bigint | 主キー                 |
| event_id     | bigint | 対象イベント           |
| leader_id    | bigint | 応募者（個人 / リーダー） |
| team_name    | string | チーム名（任意）       |

### EventTeamMember（チームメンバー）
| カラム        | 型     | 説明           |
| ------------- | ------ | -------------- |
| id            | bigint | 主キー         |
| performer_id  | bigint | 出演者応募ID   |
| member_name   | string | メンバー名     |

### EventAudience（観戦者応募）
| カラム      | 型     | 説明       |
| ----------- | ------ | ---------- |
| id          | bigint | 主キー     |
| event_id    | bigint | 対象イベント |
| user_id     | bigint | 応募者     |
